# Как создать сборку
**Перед началом работы**

Убедитесь, что:
 - Техкарта создана (например, «Хлеб пшеничный»)
 - Все указанные в ней товары есть на складе
 - Выбраны правильные склады списания и поступления (где есть сырье и куда поступит готовая продукция)

## Как создать сборку по техкарте
1. Перейдите в раздел **Фабрика → Сборки**.
2. Нажмите **➕ Сборка**. Откроется окно для добавления информации
<img src="/img/knowledge/factory/assembly-bom.png" alt="Создание сборки" width="600" />

3. В поле **Техкарта** выберите нужную — например, *Хлеб пшеничный*.
После выбора техкарты система автоматически подставит состав: сырье, нормы и готовый продукт.
4. Укажите дату сборки (подставляется текущая, при необходимости измените).
5. Введите объем сборки — например, 1 (шт., кг).
При изменении объема сборки система автоматически пересчитает количество сырья.
6. Выберите:
 - Филиал, если их несколько
 - Склад готовой продукции, куда поступит товар
 - Склад сырья, откуда будут списаны материалы
7. Проверьте заполненные данные в блоках:
 - Готовая продукция — название, количество, доля себестоимости
 - Сырье — все ингредиенты и нормы из техкарты
8. При необходимости добавьте комментарий (например, примечание о партии или качестве сырья)
9. Установите статус — например, *Завершен*
10. Нажмите **Сохранить**
