# Как добавить товары из других организаций

Если в вашей системе несколько организаций, их списки товаров могут отличаться.  
Некоторые товары присутствуют только в одной организации, а в другой — скрыты.  
iBox позволяет гибко управлять тем, какие товары видны в каждой организации.

## Почему нельзя просто создать товар заново

Иногда пользователи пытаются добавить товар вручную во вторую организацию и получают сообщение об ошибке.  

Это происходит потому, что товар уже существует в базе данных, даже если он не отображается в списке другой организации. Повторное создание приведет к дублированию и ошибкам при расчетах.


##  Как добавить товар в другую организацию
1. В правом верхнем углу откройте **меню выбора организации** и выберите **Управление**  
<img src="/img/knowledge/goods/goods-organizations2.png" alt="Управление организациями" width="500" />

2. В боковом меню перейдите в **Товары** и выберите нужный товар

<img src="/img/knowledge/goods/goods-organizations1.png" alt="Выбор товара" width="500" />

3. Нажмите **⚙️ (шестеренку)** справа от названия. Откроется окно для выбора организаций 

<img src="/img/knowledge/goods/goods-organizations.png" alt="Настройка организации" width="500" />

4. Отметьте галочками организации, где товар должен быть доступен  
5. Нажмите **Сохранить**. Теперь выбранный товар появится в списке товаров указанных организаций.


## Если нужно добавить несколько товаров:
1. В списке товаров выберите нужные позиции  
2. Нажмите на три точки в верхнем меню **… Больше → Добавить в организацию**. 
<img src="/img/knowledge/goods/goods-organizations3.png" alt="Добавить в организации" width="500" />

3. Выберите организации, в которые хотите добавить товары  
4. Нажмите **Применить**

Чтобы скрыть товары из организаций, используйте ту же кнопку:  
**⋯ Больше → Удалить из организации**, и отметьте, откуда нужно убрать доступ.

##  Автоматическое добавление новых товаров

Чтобы новые товары автоматически становились доступны во всех организациях:

1. Перейдите в **Управление организациями → Настройки → Глобальные настройки**

2. Активируйте опцию *Автоматически прикреплять новые товары ко всем организациям*

<img src="/img/knowledge/goods/global-settings.png" alt="Глобальные настройки" width="500" />

Это удобно, если все организации используют общий ассортимент и не требуют ручной настройки.


## Добавление товаров прямо из организации

Вы также можете добавить товары из другой организации напрямую:

1. Перейдите в модуль **Товары** и откройте список  
2. В верхней части справа нажмите **⋯ Больше → Добавить из других организаций**  

<img src="/img/knowledge/goods/goods-organizations4.png" alt="Добавление товаров из организаций" width="500" />

Откроется окно **Добавить (прикрепить) Товары и услуги** 

3. Выберите товары, которые хотите добавить 
4. Нажмите **Сохранить**

:::info Информация
Добавлять товары через *Управление организациями* может только администратор.  
Внутри организации это может делать любой сотрудник с соответствующим разрешением.
:::

Чтобы выдать сотруднику доступ, перейдите **Настройки → Роли → Товары → Прочее → Добавить из других организаций**.
