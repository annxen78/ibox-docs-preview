# Уведомления и заказы через Telegram

Телеграм-бот iBox — это удобный способ получать уведомления и работать с заказами, не заходя в систему.  
Существует два типа бота:

- **Бот для клиентов**
- **Бот для сотрудников**

Они выполняют разные задачи и подключаются отдельно.

## Возможности для клиентов

Клиент может

- Зарегистрироваться по номеру телефона  
- Оформлять онлайн-заказы прямо в Telegram  
- Получать уведомления о заказах, оплатах и возвратах  
- Просматривать баланс  
- Скачивать выписку в *PDF* или *Excel*  
- Получать напоминания о задолженностях  


## Возможности для сотрудников

Сотрудники получают системные уведомления в Telegram:

- Закупки  
- Заказы  
- Продажи  
- Поступления  
- Расходы  
- Возвраты  

Уведомления отправляются в рабочие чаты — не нужно открывать систему.


# Как подключить Telegram-бот

## Шаг 1. Получите ссылку на бот

### Через администратора или менеджера

Запросите ссылку на нужный бот:

- **Бот для клиентов** — уведомления, баланс, заказы  
- **Бот для сотрудников** — уведомления в рабочих группах  

### В iBox

1. Перейдите в **Настройки → Telegram-бот → Бот для контрагентов**  
   <img src="/img/knowledge/clients/telegram-bot1.png" alt="Бот для контрагентов" width="500" /> 

2. Откройте ссылку в Telegram  
   
3. Нажмите **Сохранить**  
   

## Шаг 2. Зарегистрируйтесь

### Для клиентов

1. Перейдите по ссылке на бот
<img src="/img/knowledge/clients/telegram-link.png" alt="Ссылка на бот" width="500" /> 

2.  Нажмите на **Старт**. Бот отправит вам сообщение с запросом номера телефона
3. Отправьте номер телефона, зарегистрированный в iBox  
<img src="/img/knowledge/clients/telegram-bot2.png" alt="Запуск бота" width="300" /> 

4. Дождитесь подтверждения регистрации от бота.  
  

### Для сотрудников
**Создайте бота через BotFather**
1. Откройте Telegram
2. Найдите официальный бот: @BotFather
3. Введите команду /newbot
4. Придумайте имя и username (должен оканчиваться на “bot”)
5. BotFather выдаст вам токен бота — сохраните его
6. Вставьте токен в iBox

**Вставьте токен в iBox**
1. Перейдите в **Настройки → Телеграм бот → Бот для сотрудников**
2. Вставьте токен бота
<img src="/img/knowledge/clients/token.png" alt="Токен бота" width="300" /> 

3. Сохраните настройки
После этого iBox сможет отправлять уведомления от имени вашего бота.

**Создайте Telegram-группу для сотрудников**
1. В Telegram нажмите **Создать группу**
2. Добавьте в нее нужных сотрудников
3. Назовите группу (например: *iBox: уведомления магазина*)

**Добавьте бота в группу**
1. Откройте созданную группу
2. Нажмите **Добавить участника**
3. Найдите своего бота по *username*
4. Добавьте его
5. Настройте уведомления

## Шаг 3. Настройте уведомления

После регистрации бот отправляет:

- уведомления о заказах, оплатах и возвратах  
- напоминания о задолженности  
- данные о балансе и отчеты по запросу
