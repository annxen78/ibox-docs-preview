# Добавление клиентов
1. Если вы уже подготовили список клиентов в Excel, перейдите в **Клиенты→Клиенты** и нажмите на три точки в правом верхнем углу экрана. Откроется окно для загрузки файла.

<img src="/img/knowledge/purchase/import-excel.png" alt="Добавление файла" width="600" /> 

3. В открывшемся окне **Импорт контрагентов** отсортируйте данные

<img src="/img/knowledge/purchase/import-excel2.png" alt="Добавление файла" width="600" /> 

4. Нажмите **Начать импорт**

## Чтобы добавить клиентов вручную
1. Зайдите в модуль **Клиенты**, нажмите на ➕ в правом верхнем углу. Откроется окно для добавления информации.

<img src="/img/knowledge/clients/clients-add1.png" alt="Добавление клиента" width="600" />

2. Заполните основные данные 
   - Название организации  
   - Страну и адрес  
   - Основной номер телефона  
3. Добавьте контактные данные и реквизиты 
4. Во вкладке **Маршруты** выберите, если нужно:
- Маршрут, по которому обслуживается клиент 
- Дни недели

5. Во вкладке **Прочее** укажите дополнительные данные. Данные поля не обязательны, но позволяют точнее настроить условия работы.

| Поле | Назначение |
|------|-------------|
| **Прайс-лист для клиента** | Определяет цены, по которым клиент будет получать товары |
| **Кредитный лимит** | Максимальная сумма задолженности, доступная клиенту |
| **Период консигнации** | Количество дней, в течение которых клиент может реализовать товар до оплаты |
| **Процент кешбэка** | Устанавливает скидку или бонус в процентах |
| **Периодичность отгрузок** | Интервал между поставками|
| **Сортировочный номер** | Определяет порядок отображения клиента в списке: чем выше номер, тем выше позиция|

5. Укажите владельца данных — это поможет управлять доступом и защитить клиентскую информацию

6. Нажмите **Сохранить**. Клиент появится в общем списке и будет доступен для оформления заказов и маршрутов.

:::info Информация
 - Если не указать прайс-лист, применяется прайс-лист по умолчанию  
 - Кредитный лимит блокирует продажу при превышении задолженности  
 - Период консигнации задаёт срок оплаты автоматически  
:::
