# Задачи

Раздел **Задачи** используют для планирования и контроля действий сотрудников.

## Что можно делать
- Создавать и назначать задачи на себя или других сотрудников
- Устанавливать сроки и приоритет
- Отслеживать просроченные, текущие и предстоящие задачи

## Как добавить задачу
1. Нажмите **Добавить задачу**. Откроется окно для добавления информации
<img src="/img/knowledge/crm/task.jpg" alt="Создание задачи" width="500" /> 

2. Укажите:
   - Клиента 
   - Тип задачи (звонок, встреча, сообщение)
   - Исполнителя
   - Срок выполнения
   - Комментарий

3. Нажмите **Сохранить**

После сохранения задача появится в одном из блоков:
- **Просроченные задачи** — если срок истек
- **Задачи на сегодня**
- **Задачи на завтра**

:::tip Совет
Каждая задача связана с клиентом — вы можете перейти в карточку клиента напрямую из списка задач.
:::
