# Сделки

Раздел **Сделки** позволяет управлять процессом продаж и стадиями воронки.

Каждая сделка содержит:
- Клиента и ответственного сотрудника 
- Сумму сделки
- Текущий этап 
- Историю взаимодействий

## Как создать сделку
1. Нажмите ** ➕Сделка**. Откроется окно для добавления информации.
<img src="/img/knowledge/crm/deals1.png" alt="Создание сделки" width="400" /> 

2. Добавьте:
   - Клиента  
   - Ответственного сотрудника  
   - Сумму и валюту сделки  
   - Этап (Lead, Встреча)  
3. Нажмите **Сохранить**. Сделка появится в списке.


## После создания сделки можно:
- Добавлять *заметки* — для фиксации деталей общения с клиентом 
- Прикреплять *документы* (договора, счета, коммерческие предложения) 
- Создавать *задачи* прямо из карточки сделки

<img src="/img/knowledge/crm/deal.png" alt="Сделки" width="600" /> 

:::tip Совет
Обновляйте этап сделки по мере продвижения клиента и добавляйте заметки — это поможет сохранить историю взаимодействия и повысить конверсию.
:::
