# Настройки доступа в iBox

Для настройки доступов сначала нужно создать профили сотрудников в системе, а затем назначить им роли.

## Почему важно настроить роли
Если при создании сотрудника не назначить роль, сотрудник не сможет войти в систему.

А если роль назначена неправильно — сотрудник может получить доступ к данным и действиям, которые ему не нужны.

## Работа через мобильное приложение
Даже если сотрудник будет работать только через мобильное приложение iBox, роль все равно нужно указать — хотя бы пустую.

Без роли он не сможет авторизоваться в системе и войти в приложение.
